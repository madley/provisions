Color

Color Blocks

White (#fff)

Used to bring elements forward, like buttons and form elements (radios, checkboxes, input, textarea, etc.)

Dark Gray (#2d2c2b)

This is a very dark gray used as a color block on the footer

Light Gray 1 (#f9f8f6)

Used for the main background color

Light Gray 2 (#f3f1ed)

Slightly darker than Light Gray 1, it is used for color-blocking sections, banding rows, and as a box background

Light Gray 3 (#edeae5)

Slighty darker than Light Gray 2, it is used as an accent color in areas where Light Gray 2 is the dominant color, such as in boxes, or tables as the header

Element Colors

Copy Gray 1 #857f7f

The default color used for all body copy and some headings

Copy Gray 2 #595454

This darker gray is used for emphasis and can be seen on buttons, bolded elements and headings

Copy Gray 3 #4b4647

This gray is used in conjunction with Copy Gray as a button gradient

Subtle Gray #cfceca

Used for subtle elements such as on the Back to Top button and on some headings

Mobile Green #92bc02

Used as the main call-to-action and active color

Dark Mobile Green #729d00

Used in conjunction with Mobile Green as a button gradient

Alert Red #ed1c24

Used for alerts and errors
